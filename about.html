The Development Team

We are Year 3 Bachelor's Degree students in Electrical & Electronics Engineering at TAR UMT. The Jarvis Smart Refrigerator Assistant was developed as our Capstone Project to solve real-world food waste using Edge AI and local hardware integration.

System Specifications

Speech-To-Text (STT)
Faster Whisper (small.en)
Offline neural transcription with Silero VAD (Voice Activity Detection) and dynamic noise calibration.
Edge AI Logic (LLM)
Ollama Engine
Local AI inference running the custom Jarvis model to convert natural speech into JSON executable actions.
Hardware Controller
Arduino UNO
Serial communication handling a 16x2 I2C LCD Display and a digital Door Contact Sensor.
Text-To-Speech (TTS)
pyttsx3 Engine
Thread-safe local speech synthesis providing offline vocal feedback.
Data Storage Space
Local JSON Architecture
Ultra-lightweight local storage (`fridge.json` & `waste_log.json`). Zero cloud database dependency required.
Network Architecture
Unified HTTP Server
Local API running on port 5000 with CORS and Local Network Access (LNA) bypass capabilities.
